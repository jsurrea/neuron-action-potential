# Neuron Action Potential Simulation

Simulation of the AP on a biological neuron using different Differential Equation solvers, including Forward, Backward, Euler, Runge-Kutta 2nd and 4th order. The simulation is based on the Hodgkin-Huxley model.

## How to run

Create a virtual environment:

```bash
python -m venv venv
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the simulation:

```bash
python src/interfaz.py
```
